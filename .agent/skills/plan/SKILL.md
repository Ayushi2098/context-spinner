# Planning Skill

## Purpose

Guide AI agents in planning new features for the application.

## Workflow

1. **Understand Requirements**
   - Read user request carefully
   - Identify core functionality needed
   - Consider edge cases and user experience

2. **Architecture Review**
   - Read `TEMPLATE_ARCHITECTURE.md` to understand current structure
   - Ensure new features align with existing patterns
   - Maintain GitHub Pages compatibility

3. **Break Down Tasks**
   - Create a task list using `todo_write` tool
   - Break complex features into smaller, manageable steps
   - Identify dependencies between tasks

4. **Implementation Plan**
   - Define data structures and state management
   - Plan component hierarchy
   - Consider devtools integration (if needed)

5. **Execution**
   - Implement features following the plan
   - Update documentation as needed
   - Commit changes frequently

## Guidelines

- Keep features modular and reusable
- Follow existing code patterns
- Ensure production builds work correctly
- Test in development mode with `pnpm dev:full`
